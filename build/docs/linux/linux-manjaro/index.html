manjaro 的指令​

  • pacman:自带包管理工具

    • -S:安装

    • -Ss:搜索

    • -Syy:更改/etc/pacman.conf 添加源后 ，更新软件包数据库

    • -Syu:更改系统本身

    • -Sc:清除缓存

    • -Scc:清除已下载的安装包

    • R:删除服务,不能停顿太久

    • Rs:删除服务，以及相关依赖

    • Rns:删除服务，以及相关依赖,以及全局变量

    • Q:查询已安装程序

    • Qe:查询自己安装的程序

    • Qeq:查询自己安装的程序（不显示版本号）

    • Qs key:查询本地名字有 key 的程序

    • Qdt:不需要的软件

    • pacman -R $(pacman -Qdtq):删除不需要的软件，最后 q 去除版本号

    • whereis name 查询程序安装位置

  • /etc/pacman.conf:pacman 配置文件 添加源,

    • Color:设置高亮变色

版本更新报错​

1.首先更新一下密钥，如果没有安装 archlinux-keyring,请及时安装 sudo pacman -Syy && sudo pacman -S archlinuxcn-keyring

sudo pacman-key --refresh-keys 2.重新加载相应的签名密钥 sudo pacman-key --init sudo pacman-key --populate 3。清除 pacman 的缓冲文件 sudo pacman -Scc 4.更新或者安装系统即可 sudo pacman -Syu || sudo pacman -Syudd(-dd 跳过全部检测)_
